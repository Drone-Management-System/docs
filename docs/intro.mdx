# DMS Backend Documentation

This site provides API references and technical module documentation for DMS
backend services. API references are grouped by module so each service keeps its
endpoint surface, authentication rules, request formats, response formats, and
error behavior separate.

## API Reference

| API | Status | Entry Point | Notes |
| --- | --- | --- | --- |
| API Catalog | Active | [API Reference](./api/overview) | Module-by-module API index. |
| CAA Registry | Documented | [CAA Registry API](./api/caa-registry/overview) | Consumer-facing endpoint reference, auth rules, request examples, response shapes, and errors. |
